# ezword-python

主要用于采集 Ezword 的单词来源并入库

## 词库源

[Dict](https://github.com/kajweb/dict)
[Dictionary](https://github.com/mahavivo/english-dictionary)

### 单词发音

有道发音接口：

```bash
#type 1 为英音 2 为美音
https://dict.youdao.com/dictvoice?audio={word}&type={1|2}

```
